# MyProject

## ВНИМАНИЕ, БД НЕ ПЕРЕБРАСЫВАЕТСЯ И НЕ ХОСТИТСЯ ГДЕ-ЛИБО, КАЖДЫЙ РАБОТАЕТ СО СВОЕЙ БД, СООТВЕТСВЕННО, ОНА БУДЕТ ПУСТА ПРИ СОЗДАНИИ

Учебный блог на PHP с использованием PostgreSQL.

## Стек

- PHP 8.2 + Apache
- PostgreSQL 16
- Docker
- Docker Compose
- GNU Makefile

Приложение доступно на `http://localhost:8080`

## Команды

```bash
make up       # запустить контейнеры
make down     # остановить контейнеры
make logs     # логи веб-сервера
make bash     # войти в контейнер
```

## Маршруты

| URL | Описание |
|---|---|
| `/` | Главная страница со списком статей |
| `/articles/{id}` | Страница статьи |
| `/articles/{id}/edit` | Редактирование статьи |

## Запуск

1. Склонировать репозиторий
2. Создать `.env` файл:

DB_NAME=mydb
DB_USER=postgres
DB_PASSWORD=yourpassword

3. Запустить контейнеры:
```bash
make up
```

Приложение доступно на `http://localhost:8080`
